Java로 Calender를 구현한 예제
==================
Java와 Swing Component를 사용해서 실시간 공유 및 다중 채팅 캘리더를 구현

## Main Function
+ Sign in/Login
+ Calender
+ Multi Catting
+ Real Time Sharing to calendar
+ Random color by User

## Development Tools
 + JDK 1.8.11
 + Swing Component
 + Eclipse Neon
 + Apache Tomcat 8.0
 + Mysql
